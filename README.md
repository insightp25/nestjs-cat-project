# 고양이의, 고양이에 의한, 고양이를 위한 커뮤니티 웹애플리케이션

![Screen Shot 2022-08-01 at 3 36 06 PM](https://user-images.githubusercontent.com/71416000/182086918-0e23a205-b9b9-42c3-9e12-d3734db98703.jpg)

### 주요기능
1. 회원가입
2. 로그인/로그아웃
3. 사진 포스팅
4. 댓글 작성과 좋아요 기능

### 활용 기술
- NestJS
- Express
- Node.js
- MongoDB
- Mongoose
- Swagger
- PM2
- AWS

### 활용 개념과 지식
- 객체 지향 설계와 캠슐화, 의존성 주입
- AOP 패턴
- Repository 패턴
- 보안 인증
- HTTP
- DB 설계와 정규화
- 유닛 테스팅, 통합 테스팅
